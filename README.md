# netWatch

a full production-grade Windows network intelligence platform

Overview

netWatch is a Windows-native, high-performance, production-grade network observability and traffic control platform. It combines realtime packet capture and analysis, device discovery, topology visualization, DNS intelligence, traffic policy enforcement, and a modern Tauri/React frontend with a cyberpunk-inspired UI.

This repository contains the initial architecture and scaffolding for the project.

Key goals

- Defensive, monitoring-only network intelligence
- Native Windows integration (WFP, ETW, WMI, Npcap)
- Rust core engine for performance, safety, and concurrency
- Tauri + React TypeScript frontend for polished UX
- SQLite for local storage
